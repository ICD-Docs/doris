# Uso de DORIS desde la API de la CIE

## Para desarrolladores de software e integradores de sistemas:

Es posible acceder al motor de reglas de DORIS de forma programática a través de la API de la CIE. Actualmente, está disponible una versión preliminar de este sistema.


- [La página principal de la API de la CIE](https://icd.who.int/icdapi) y 
- [La página con documentación sobre la API de la CIE](https://icd.who.int/docs/icd-api/DORISSupport/) tienen más información sobre la API

- [En la página de referencia sobre la API de la CIE (Open API)](https://id.who.int/swagger/index.html) se ofrecen explicaciones detalladas sobre los puntos de acceso (<i>endpoints</i>) que admite la API, incluido el de DORIS.
